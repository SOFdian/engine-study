| **模块名称**      | **核心职责**                                 | **关键组件**                                                 | **依赖要求**                           |
| ----------------- | -------------------------------------------- | ------------------------------------------------------------ | -------------------------------------- |
| **docs/**         | 存储项目所有非代码文档。                     | 学习笔记、架构图、README、复盘记录。                         | 无                                     |
| **math/**         | 引擎的数学基石，提供纯数值计算。             | 向量 (Vector)、矩阵 (Matrix)、四元数 (Quaternion)、几何求交算法。 | **零依赖**。禁止依赖任何其他引擎模块。 |
| **engine_core/**  | 提供引擎运行所需的基础设施。                 | 日志系统 (Logger)、断言 (Assert)、内存分配器 (Allocator)、性能分析 (Benchmark)。 | 仅可依赖 `math`。                      |
| **renderer_cpu/** | 教学性质的软件渲染器，用于透彻理解渲染管线。 | 软光栅化、深度测试、顶点变换、纹理采样。                     | 依赖 `math` 和 `engine_core`。         |
| **renderer_gpu/** | 基于现代 API 的实时渲染框架。                | Vulkan/DX12 封装、Shader 管理、GPU 资源分配。                | 依赖 `math` 和 `engine_core`。         |
| **tools/**        | 辅助开发的独立工具模块。                     | 模型加载器 (OBJ Loader)、文件系统封装、配置解析。            | 依赖 `math` 和 `engine_core`。         |
| **tests/**        | 自动化验证所有模块的正确性。                 | 单元测试用例、集成测试、性能基准测试代码。                   | 依赖所有被测试的模块。                 |
